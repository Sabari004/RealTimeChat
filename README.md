# RealTimeChat
It is a useful webApp which is used to chat as a group without storing the messages,here we can create many rooms and join that chat

## DEPLOYMENT
- Clone this repository
- `npm install` 
- `npm start` on both client & server
- the app will run on http://localhost:3000
